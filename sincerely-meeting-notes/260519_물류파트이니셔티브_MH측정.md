### 전략적 배경

전 팀 회의 의사결정 원칙에 따라, 팀 전체의 통일된 재무 지표 기반 목표를 우선 수립하고 AX/DX 이니셔티브의 프로젝트 범위(Scope)를 사전 확정 후 추진한다. 각 파트(물류/생산/품질)의 노력이 하나의 목표로 수렴되지 못하고 범위가 불분명한 상태에서 추진될 경우, 리소스 낭비 및 매몰 비용 리스크가 발생한다는 점을 전제로 한다.

물류파트 목표는 **AX 기반 생산성 50% 향상**이며, 이를 측정하기 위한 현재 WMS+TMS 가동률은 **20.8%** (184.6 MH / 886 MH 가용, W21 기준)이다. 피킹·상하차·외주임가공·프리패키징 공정은 아직 미포함 상태이므로 실제 가동률은 더 높을 것으로 추정되며, M/H 기준선 확립이 50% 목표 달성 여부 판단의 선행 조건이다.

## 3. 안건별 논의 및 의사결정 (Key Discussions & Decisions)

### 안건 1: 문경선 Cycle time 실측 데이터 수신 및 CBM 비교 검증

- **논의 배경:** 현재 WMS M/H 산출은 글로벌 CBM 상수(4.0 min/CBM)에 의존하고 있어 자사 실제 공정 시간과의 괴리가 존재한다. 문경선 파트원이 입하~입고~검수 구간을 수기 Cycle time 방식(시작~끝 wall-clock time)으로 측정 중이며, 해당 데이터를 수령 후 CBM 상수와 비교 검증한다.
- **의사결정 결과:** 문경선이 측정 완료 즉시 데이터를 공유하고, 윤지수/Claude가 수신 후 CBM 상수 대비 비교 분석을 실행한다. 검증 완료 시 자사 실측 상수로 교체(Iter 2.0 선행 작업).
- **결정 근거 / 기대 효과:** 글로벌 상수 대비 자사 실측 상수로 교체 시 M/H 정확도 30~50% 향상 기대. 기준선 신뢰도 확보는 50% 생산성 향상 목표 달성 여부 판단의 전제 조건.

### 안건 2: WMS movement 테이블 M/H 필드 백필 및 프리패키징 연동

- **논의 배경:** 생산운영팀은 pkg_schedule 3-table ELS(예상M/H·계산M/H·실측M/H 구조)를 이미 운영 중이다. WMS 입하 베이스(appLui4ZR5HWcQRri) movement 테이블에는 M/H 필드가 아직 없어 공정별 실적 집계가 불가능한 상태다.
- **의사결정 결과:** 입하(Receiving), 입하검수(Inbound QC), 입고(Putaway), 출고검수(Outbound QC) 4개 공정에 예상M/H·계산M/H·실측M/H 필드를 추가한다. 프리패키징 공정도 동일 movement 레코드 측정 대상에 포함한다.
- **결정 근거 / 기대 효과:** 생산운영팀 ELS 구조와 통일함으로써 파트 간 M/H 집계 방식의 일관성을 확보. 추후 통합 대시보드 구성 시 데이터 정합성 확보 가능.

### 안건 3: 프리패키징 M/H 측정 설계 착수

- **논의 배경:** 프리패키징 공정은 DEFERRED 상태로 미측정 구간이었다. 글로벌 벤치마크(WERC 등)에 해당 공정 표준이 없으며, 사내 고유 공정이므로 직접 타임 스터디 설계가 필요하다.
- **의사결정 결과:** 타임 스터디 착수. 대상 요소: MES 연동 인쇄 준비 / 박스 성형 / 라벨 부착 / 스테이징. 제안 공식: M/H = (인쇄준비 + 박스성형×qty + 라벨부착×qty + 스테이징×carton_qty) × PFD(1.15). 출고수량·임가공 예정일 필드는 기존 존재 — 실측 완료 즉시 연동 가능.
- **결정 근거 / 기대 효과:** 프리패키징은 현재 M/H 미계산 공정으로, 실측 완료 시 전체 물류 가동률 기준선 상향 조정 가능. PFD (Personal + Fatigue + Delay) 계수 1.15 적용으로 피로·지연 여유율을 반영한 현실적 표준 시간 확보.

### 안건 4: TMS M/H 측정 실측 착수

- **논의 배경:** 현행 TMS M/H는 글로벌 표준값(배차 1.0분 + 서류 2.0분) × PFD(1.15) = 3.45분/건을 그대로 사용 중이다. W21 기준 321건 / 18.5 MH이며, 상차(Loading), 외주임가공 준비(다영기획 Prep) 등 미설계 공정이 존재한다.
- **의사결정 결과:** TMS 글로벌 상수를 자사 실측 기반으로 교체하는 Iter 착수. 상차·외주임가공 준비 공정을 단계별로 추가 설계한다.
- **결정 근거 / 기대 효과:** 글로벌 상수 기반 수치의 정확도 불확실 — 실측 교체 시 TMS M/H 정밀도 향상 및 운송비 원가 산출 신뢰도 제고. 미설계 공정 포함 시 전체 TMS M/H 재산출 기준 확립 가능.

### 안건 5: 통합 M/H 대시보드 — 추후 결정

- **논의 배경:** WMS(appLui4ZR5HWcQRri)와 TMS(app4x70a8mOrIKsMf) 베이스가 현재 분리 운영 중이며, CBM 기반(WMS) vs 건수 기반(TMS) 집계 방식이 상이하다.
- **의사결정 결과:** 데이터 정합성 확보(Iter 2.x 완료) 이후 통합 뷰 형태를 결정한다. 현 시점 통합 시 오산 위험 존재 — 보류.
- **결정 근거 / 기대 효과:** 조기 통합 시 집계 기준 불일치로 인한 M/H 오산 리스크 발생. Iter 2.x 완료 후 통합 시 정확도 보장 가능.
