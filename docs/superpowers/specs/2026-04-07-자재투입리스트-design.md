## Context

베스트원 외부 창고 → 에이원지식산업센터 자재 보충 프로세스를 현재 Airtable 뷰에서 수동으로 확인하고 있음.  
사용자가 `/투입리스트` 커맨드를 입력하면 Claude가 MCP로 Airtable을 조회해 이동 지시 초안을 채팅으로 출력하는 스킬을 구현한다.

**두 가지 자재 흐름:**
1. **조립투입 재고자재** — 베스트원 창고 재고 → 에이원센터 (조립 작업용)
2. **생산투입 신시어리웨일즈** — 신시어리웨일즈 인쇄 후 → 에이원센터 입하

## 보충수량 계산 공식

```
보충수량 = MAX(0, 발주요청수량 - 에이원현재고 + 100)
```

- **발주요청수량**: `발주요청수량(고객주문×단위수량)` 필드 (movement 테이블, hidden 필드 — 구현 시 ID 확인 필요)
- **에이원현재고**: material(parts-stock) 또는 sync_parts의 재고위치별 수량에서 에이원 분리값 (구현 시 필드 확인 필요)
- **안전재고**: 100개 고정 (추후 파라미터화 가능)
- 보충수량 ≤ 0 이면 → "보충불필요" 표시
