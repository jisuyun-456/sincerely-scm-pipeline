03 · 의사결정 (D-01 ~ D-05)
D-01
Cycle Time 실측 데이터 수신 → CBM 비교 검증
문경선이 수기 측정 중인 입하~입고~검수 Lead time을 수령하여 현행 CBM 글로벌 상수(4.0 min/CBM)와 비교 분석. M/H 정확도 30~50% 향상 기대.
연계: Iter 2.0 선행 작업
D-02
WMS movement 테이블 M/H 필드 백필
4개 공정(입하·입하검수·입고·출고검수)에 예상M/H · 계산M/H · 실측M/H 3-필드 추가. 생산운영팀 pkg_schedule ELS 구조와 통일하여 파트 간 집계 일관성 확보.
연계: Iter 1.6
D-03
프리패키징 M/H 타임 스터디 착수
MES 인쇄준비 + 박스성형 + 라벨부착 + 스테이징 4요소 직접 실측. 공식: M/H = (인쇄준비 + 박스성형×qty + 라벨부착×qty + 스테이징×carton_qty) × PFD 1.15
연계: Iter 2.3
D-04
TMS M/H 실측 기반 상수 교체 Iter 착수
현행 3.45 min/건 글로벌 표준(배차 1.0 + 서류 2.0 × PFD) → 자사 실측값으로 교체. 상차·외주임가공 준비 공정 단계별 추가 설계.
연계: 단기 착수
D-05
통합 M/H 대시보드
WMS(CBM 기반) + TMS(건수 기반) 집계 방식 상이. 조기 통합 시 오산 위험 — Iter 2.x 완료 후 통합 형태 결정.
연계: Iter 2.x 완료 후
